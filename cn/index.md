# Constellar Studio 星合工作室

星合工作室致力于利用卫星与机载雷达数据研究地表变化。我们提供基础设施监测、地表形变与变化检测，以及地球内部结构三维重建等相关服务。

我们的工作涵盖多源合成孔径雷达（SAR）数据处理，包括卫星与机载平台数据。我们分析不同空间与时间尺度下的地表形变，支持识别道路、桥梁、建筑等关键基础设施的潜在风险。同时，我们也为野火、洪水、滑坡以及多年冻土退化等自然灾害提供早期监测与环境咨询服务。

通过对雷达数据的系统分析，我们希望为工程建设、环境保护与土地管理提供决策支持，降低风险，并加深对自然环境与人类活动区域变化的理解。

## 我们可以做什么

- 利用卫星与机载 SAR 数据分析地表变化  
- 多时空尺度下的地表形变监测与变化检测  
- 道路、桥梁、建筑等关键基础设施风险识别  
- 野火、洪水、滑坡及多年冻土等自然灾害监测与环境咨询
