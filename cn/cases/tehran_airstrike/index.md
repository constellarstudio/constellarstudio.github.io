# 基于多时相 SAR 变化检测的德黑兰城市空袭损毁识别

近期，德黑兰部分城区在空袭事件中遭受破坏，引发了对受损区域空间分布与破坏程度的关注。由于冲突环境下地面信息获取往往受限，合成孔径雷达（SAR）遥感凭借其全天时、全天候观测能力，为灾害与冲突事件后的快速损毁评估提供了重要技术手段。本研究基于多时相 SAR 影像开展变化检测分析，旨在通过雷达信号特征识别潜在的结构性破坏区域，并与公开报道中的受损地点进行空间对比验证，以评估该方法在复杂城市环境中的应用效果。

本研究针对德黑兰近期空袭事件开展了基于合成孔径雷达（SAR）的变化检测分析，并重点对新闻报道中提及的几处受损区域进行了放大核查。这些区域包括阿扎迪体育综合体、甘地医院、莫塔哈里医院以及古列斯坦宫周边区域。通过对变化检测结果进行高分辨率目视检查，可以在上述区域中清晰识别到显著变化信号，在图中以黄色标示。

![局部放大核查结果](/assets/images/tehran_airstrike/small_map.webp)

其中，阿扎迪体育综合体的变化主要集中在体育场看台建筑区域，表现为局部结构受损；甘地医院与莫塔哈里医院周边则呈现出范围更大的变化斑块，显示出较为广泛的结构扰动；而古列斯坦宫附近的变化信号则相对局部，表明该区域可能仅发生了部分损伤。总体而言，变化检测结果与新闻报道所描述的受损区域在空间上具有较好一致性，说明本项目所采用的方法能够有效识别潜在的空袭破坏区域，达到了预期的分析目标。

在具体处理流程上，本研究针对德黑兰城市环境中 SAR 变化检测面临的核心问题进行了专门设计，即如何区分由建筑物破坏造成的真实变化与由植被生长、地形起伏或平滑地表引起的伪变化。首先，通过结合干涉相干度与数字高程模型（DEM）构建低相干城市掩膜，并将分析范围限制在海拔 1500 米以下且相干度低于 0.4 的区域，从而有效排除了德黑兰北部山区中由植被覆盖导致的自然相干失稳区域。随后，以初始时相的后向散射强度作为稳定性判定基准，将幅度值高于 −5 dB 的像素定义为稳定目标区域。这一步骤能够进一步识别并剔除如机场跑道或开阔平地等地表，这类区域虽然可能表现出较低相干度，但其物理结构在观测期间通常保持稳定，因此不应被误判为变化区域。

在变化强度的判定方面，本研究采用跨时段雷达幅度差值的绝对值作为主要指标，并设置 3 dB 作为变化检测阈值。这一阈值的选择在一定程度上平衡了 SAR 影像固有的相干斑噪声与真实地物变化所产生的信号波动。在干燥的德黑兰城市环境中，建筑结构一旦遭到破坏，往往会打破原有的二次散射几何结构，从而导致后向散射强度出现明显变化。简单来说，相干度主要反映雷达信号的相位稳定性，而幅度则反映地表散射强度的变化。当爆炸导致建筑倒塌、碎片堆积、金属结构暴露或地表粗糙度发生变化时，雷达后向散射强度往往会出现显著增加或减少。

![德黑兰大范围幅度变化检测图](/assets/images/tehran_airstrike/large_map_ampchange.webp)

最终的变化检测结果通过对低相干掩膜与幅度变化掩膜进行空间交集运算得到。这种多准则耦合的检测策略显著提高了结果的可靠性，因为只有同时满足相干性丧失和散射强度显著变化的像素才会被识别为潜在变化区域。这样的保守提取方法能够有效压制由环境因素或随机噪声引起的虚警，从而更准确地锁定可能发生结构性破坏的区域。

此外，空袭事件对地表造成的破坏通常具有一定的空间连续性。尽管在处理过程中采用了多视（multi-looking）操作，使得影像的空间分辨率有所降低，但典型的空袭损毁区域在影像中仍然往往表现为由数个至数十个连续像素组成的变化斑块。这种空间聚集特征使真实的损毁信号能够从孤立的系统噪声中区分出来。对于结果中出现的一些未被新闻报道证实的零散变化点，虽然其信号特征符合检测模型所定义的变化条件，但仍需考虑城市内部正常的人类活动，例如施工工程、大型车辆移动或临时设施变化等因素的影响。总体而言，这种基于雷达物理特性的多准则筛选方法，为在复杂城市环境中快速识别大范围结构性破坏提供了一种稳健而有效的遥感分析手段。
