# 多源遥感下的京津冀极端暴雨洪涝监测

七月下旬，京津冀地区受副热带高压边缘暖湿气流与台风“竹节草”共同影响，持续遭遇强降雨。

截至7月26日18时，河北省保定市阜平县降雨量已相当于当地平常一整年的总和。暴雨造成4万余人受灾，2人遇难，2人失踪。卫星图像显示，城区河道水位暴涨，洪水冲出河道，涌入周边区域，部分低洼道路被完全淹没。

![阜平县洪水淹没对比](/assets/images/bth_flood/Fuping.webp)

河北省承德市兴隆县自7月23日起连降暴雨，引发山洪，冲毁桥梁道路，冲走房屋，全县已有8人遇难、18人失联。卫星图像显示，兴隆县城内涝严重，主干道兴隆大街出现明显积水。由于兴隆县地处林区，云层厚重，光学影像受影响较大，地面观测难度随之增加。

![兴隆县内涝](/assets/images/bth_flood/Xinglong.webp)

27日晚至28日凌晨，来自兴隆县的山洪西流汇入清水河，使清水河流量飙升至常态的1500倍。洪水沿河道席卷北京密云区北庄镇和太师屯镇，最终注入密云水库。卫星图像显示，清水河原有河道被大幅拓宽，沿岸村镇严重受淹。北庄镇搜狐农场被洪水吞没，太师屯镇葡萄园村一家养老院被洪水冲毁，造成31人遇难。

![密云区清水河洪水范围](/assets/images/bth_flood/Miyun.webp)

在本次案例中，我们采用了Sentinel-1雷达影像与Sentinel-2光学影像，均为高分辨率卫星遥感数据。通过对比暴雨前后SAR影像幅度变化，可识别部分洪水淹没区域；而光学影像则通过计算归一化水体指数（NDWI）来提取水体。

结果显示，光学影像在洪涝区域探测上精度更高，而SAR仅能识别部分区域。其原因可能在于河流周边土壤原本含水量较高，导致暴雨前后幅度值变化不显著。未来可考虑引入相干性系数，以提高SAR洪水监测能力。

同时，光学影像易受云层遮挡。洪灾属于突发性强、持续时间短的灾害，往往难以获取足够的无云影像，限制了其应用效果。相比之下，SAR不受天气和光照条件制约，具备全天候、全天时的观测优势，是洪水监测的重要补充。结合多源遥感数据，可显著提升洪涝灾害识别的精度与效率。
